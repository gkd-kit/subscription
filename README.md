# @gkd-kit/subscription

重要说明: 本人审核精力有限，所有规则均暂时停止维护, 你可以自行编写规则, 导入本地订阅使用

---

`默认规则` 仅启用 `开屏广告` 一类规则, 其它所有规则均需用户手动打开

在 GKD 内添加以下链接即可使用此规则

```txt
https://fastly.jsdelivr.net/npm/@gkd-kit/subscription
```
